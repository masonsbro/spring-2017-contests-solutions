Seat Numbers

I was preparing to board my flight. Unfortunately, I lost my boarding card! Of course, my biggest concern is the size of the aircraft I'm about to fly on. I remember my seat number, however. Given my seat number, can you tell me the minimum possible capacity of my aircraft?

My seat number is of the form [row number][aisle letter]. Aisles in this aircraft are guaranteed to be labeled as A-F. There are no incomplete aisles.

Input

The first line contains an integer T. T lines follow. Each line contains a string of the form [N][L].

Output

Please output T lines where the ith line contains the answer to the ith test case.

Constraints

1 <= T <= 10
1 <= N <= 1000000000
'A' <= L <= 'F'
1 <= i <= T

Sample Input

3
23A
46F
1B

Sample Output

138
276
6

Explanation

For the first case, there must be at least 23 rows in the aircraft, where each row can seat 6 passengers.
