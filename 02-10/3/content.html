Flight Numbers

After going through security, you stop by the arrivals/departure monitors to find your gate. You don't remember your flight number, but you remember that the XOR of its bits (when written in binary) is 1.

Given all the flight numbers on the monitor, how many of them might be your flight?

Input

The first line will contain a single integer T, the number of test cases. T test cases follow.

The first line of each test case will contain a single integer N, the number of flights. The second line of each test case will contain a list of N distinct space-separated integers X_i, the flight numbers.

Output

Your program should output T lines, one for each test case.

For each test case, output the number of flight numbers X_i such that the XOR of the bits in X_i is 1.

Constraints

1 <= T <= 100
1 <= N <= 1,000
1 <= X_i <= 1,000,000,000

Sample Input

1
5
1 5 4 2 3

Sample Output

3

Explanation

Below are the binary representations of the flight numbers:

  • 1 = 0001b
  • 5 = 0101b
  • 4 = 0100b
  • 2 = 0010b
  • 3 = 0011b

Below are the XORs of the bits in the flight numbers:

  • 0001b => 1
  • 0101b => 0
  • 0100b => 1
  • 0010b => 1
  • 0011b => 0

So the XORs of the bits of 3 of the flight numbers are 1.
